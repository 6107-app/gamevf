## 🌊 What is DeepCast?

DeepCast is a **4-player competitive fishing game** built on Ethereum. Players bet ETH, cast their rods, and compete for the highest fish score. Observers can watch live and bet on who wins — just like a prediction market.

**The twist:** every fish's rarity, weight, and dice roll is decided by Chainlink VRF on-chain. No server. No manipulation. Anyone can verify every result on the blockchain.

```
🎮 4 players enter a room  →  💰 Each bets ETH  →  🎣 Everyone fishes simultaneously
        ↓
🎲 Chainlink VRF decides the fish (rarity + weight)
        ↓
⚖️ Scores calculated on-chain  →  🏆 Winner takes the pot
        ↓
👁️ Spectators bet on who wins via Prediction Market
```

## 🚀 Quick Start

### Prerequisites

Make sure you have these installed:

```bash
# Node.js 18+
node --version

# Foundry (Forge + Anvil + Cast)
curl -L https://foundry.paradigm.xyz | bash
foundryup

# Check versions
forge --version
anvil --version
```

### 1. Clone the repo

```bash
git clone https://github.com/6107-app/gamevf.git
cd gamevf
```

### 2. Install dependencies

```bash
# Root dependencies
npm install

# Frontend dependencies
cd frontend && npm install && cd ..

# Contracts: install Foundry submodules
cd contracts && forge install && cd ..
```

### 3. Set up environment variables

```bash
cp frontend/.env.example frontend/.env.local
```

Edit `frontend/.env.local`:

```env
# Local development (Anvil)
NEXT_PUBLIC_RPC_URL=http://127.0.0.1:8545
NEXT_PUBLIC_CHAIN_ID=31337

# Fill these in after deploying contracts (Step 5)
NEXT_PUBLIC_GAME_ADDRESS=0x...
NEXT_PUBLIC_ROD_ADDRESS=0x...
NEXT_PUBLIC_PREDICTION_MARKET_ADDRESS=0x...

# Optional: set to 'true' to simulate transactions without sending
NEXT_PUBLIC_SIMULATE_TX=false
```

### 4. Start the local blockchain

```bash
# Terminal 1 — Start Anvil local node
anvil
```

Anvil starts at `http://127.0.0.1:8545` with 10 pre-funded test accounts.

### 5. Deploy contracts

```bash
# Terminal 2
cd contracts

# Build contracts
forge build

# Deploy to local Anvil
forge script script/Deploy.s.sol \
  --rpc-url http://127.0.0.1:8545 \
  --private-key 0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80 \
  --broadcast
```

Copy the deployed contract addresses into your `.env.local`.

### 6. Start the VRF auto-fulfillment bot

```bash
# Terminal 3 — automatically fulfills VRF requests on local chain
cd contracts

# Create .env.local for the bot
cp .env.example .env.local
# Fill in: RPC_URL, PRIVATE_KEY, VRF_ADDRESS, GAME_ADDRESS, ROD_ADDRESS

node --env-file=.env.local scripts/autoFulfill.js
```

### 7. Start the frontend

```bash
# Terminal 4
cd frontend
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) 🎉

## 🎮 How to Play

### As a Player

1. **Connect MetaMask** — switch to Sepolia testnet
2. **Buy a fishing rod** — go to Equipment Hall, mint a rod NFT
3. **Join or create a room** — choose your tier (Bronze starts at 0.01 ETH)
4. **Wait for 4 players** — host clicks Start when ready
5. **Cast your rod** — click the screen to cast
6. **Reel it in** — press `Space` when the bird lands on the green zone
7. **Decide** — lock in your fish or bet more and re-cast (up to 3 times)
8. **Collect winnings** — top 3 players share the pot

### As a Spectator

1. Find a **Live** room in the lobby
2. Watch real-time player stats
3. Place a bet on who you think will win
4. The earlier you bet, the higher your time-weight multiplier (up to 1.5×)
5. Collect your share of the spectator pot if you guessed right

## 🎣 Fish Rarity & Scoring

```
Score = Rarity × Weight × Time Bonus × Recast Bonus × Dice Multiplier
```

| Rarity | Score × | Probability | Weight Range |
|---|---|---|---|
| ⚪ Common | ×1 | 55% | 0.05 – 1.5 kg |
| 🟢 Uncommon | ×2 | 25% | 0.3 – 4.0 kg |
| 🔵 Rare | ×4 | 13% | 1.5 – 8.0 kg |
| 🟣 Epic | ×8 | 6% | 4.0 – 20.0 kg |
| 🟡 Legendary | ×16 | 1% | 6.0 – 25.0 kg |

**Time Bonus:** faster = better (≤60s → ×1.2, >120s → ×0.7)

**Re-cast Bonus:** more bets = bonus multiplier (4 bets → ×1.15)

## 🗂️ Project Structure — Detailed

```
contracts/src/
├── FishingGame.sol
│   ├── createRoom() / joinRoom() / startGame()
│   ├── cast()           — triggers Chainlink VRF request
│   ├── lockIn()         — finalizes player's fish result
│   ├── recast()         — re-bet + triggers dice VRF
│   ├── fulfillRandomWords() — VRF callback: generates fish + dice
│   ├── _calculateScore()    — Rarity × Weight × Time × Recast × Dice
│   └── _settleGame()        — ranks players, distributes prizes
│
├── FishingRod.sol
│   ├── mintRod()        — ERC-721 mint with ETH payment
│   ├── upgrade()        — VRF-based stat upgrade
│   ├── repairFull/Partial() — restore durability with ERC-20 tokens
│   └── consumeDurability()  — called by FishingGame after each cast
│
└── PredictionMarket.sol
    ├── openBetting()    — opens spectator market for a room
    ├── placeBet()       — spectator bets ETH on a player
    ├── triggerSettle()  — called by FishingGame on game end
    └── claimPrize()     — winner spectators claim rewards
```
